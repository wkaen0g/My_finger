# 20BN-Jester V1 官方完整下载教程（Windows 版）

## 一、基础信息
- 用途：手势识别模型预训练（适配你当前的“手势控制电脑”项目）
- 官方完整体积：约 22.8 GB（远小于 50GB 限制）
- 安装路径：`G:\20BN-Jester V1`（全程不走 C 盘）
- 包含内容：
  - 148092 个手势视频帧序列（27 类手势）
  - 官方训练/验证/测试标签 CSV 文件

---

## 二、前置准备
1. 新建目标目录（全部放在 G 盘）：
G:\20BN-Jester V1\downloads # 存放官方分卷压缩包
G:\20BN-Jester V1\csv # 存放标签文件
plaintext

2. 安装 7-Zip（推荐，解压大文件更快）：
- 官网下载：https://www.7-zip.org/
- 安装后右键菜单会出现“7-Zip”选项

---

## 三、官方 Qualcomm 视频数据下载
### 1. 进入官方下载页面
1. 访问 Qualcomm Jester 数据集主页：
https://developer.qualcomm.com/software/ai-datasets/jester
2. 登录/注册 Qualcomm 开发者账号（个人邮箱即可）
3. 进入「Downloads」标签页，你会看到：
- `Jester dataset download instructions`（官方说明文档）
- `Jester_zip1` / `Jester_zip2` / `Jester_zip3`（视频分卷压缩包）

### 2. 下载文件到 G 盘
1. 先点击下载 `Jester dataset download instructions`，保存到 `G:\20BN-Jester V1\downloads`
2. 依次下载 `Jester_zip1`、`Jester_zip2`、`Jester_zip3`，全部保存到 `G:\20BN-Jester V1\downloads`
- 三个分卷合计约 22.8GB，全程不走 C 盘

### 3. 合并与解压分卷
1. 打开 PowerShell，进入下载目录：
```powershell
cd "G:\20BN-Jester V1\downloads"
合并三个分卷为一个完整文件：
powershell
cmd /c "copy /b Jester_zip* Jester_full.zip"
用 7-Zip 解压 Jester_full.zip：
右键 Jester_full.zip → 7-Zip → 提取到当前目录
解压后会得到 20bn-jester-v1 文件夹，里面是 1~148092 视频帧目录
移动到最终目录：
powershell
move "G:\20BN-Jester V1\downloads\20bn-jester-v1" "G:\20BN-Jester V1\"
四、标签文件下载（备用方案）
如果官方标签文件无法下载，直接用 Kaggle 备用版，和官方数据完全兼容：
1. 网页直接下载（推荐）
打开 Kaggle 标签数据集页面：
https://www.kaggle.com/kylecloud/jester-csv
点击右上角 Download，选择 Download dataset as zip (701 kB)
保存压缩包到 G:\20BN-Jester V1\csv
2. 解压标签文件
进入 G:\20BN-Jester V1\csv，右键压缩包 → 7-Zip → 提取到当前目录
解压后会得到 3 个关键文件：
jester-v1-train.csv
jester-v1-validation.csv
jester-v1-test.csv

七、常见问题与解决
下载分卷失败 / 卡住：
换用 Edge/Chrome 浏览器重试，或使用 IDM 多线程下载
合并解压报错：
确保 3 个分卷都下载完整，文件名未被修改
标签文件和视频 ID 不匹配：
必须使用本教程的 Kaggle 备用标签文件，与官方数据完全对应
C 盘被占用：
全程操作都在 G:\20BN-Jester V1 目录下，C 盘只会产生几 KB 缓存
